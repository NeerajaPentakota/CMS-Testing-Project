# Content-Management-Systems-Journal-Testing-project
# CMS Journal Testing Project

## Objective
This project aims to perform end-to-end testing of a Journal/Content Management System using WordPress.

## Scope
- Functional Testing
- API Testing
- Database Testing
- Security Testing
- Accessibility Testing
- Regression Testing

## Tools Used
- WordPress
- Jira
- Postman
- MySQL Workbench
- GitHub
- Selenium IDE

## Modules Covered
- Authentication
- Article Submission
- Metadata Validation
- Editorial Workflow
- Media Upload
- REST APIs
